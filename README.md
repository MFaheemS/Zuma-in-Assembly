# Zuma-in-Assembly
console-based Zuma-inspired arcade game implemented entirely in x86 Assembly language (MASM) using the Irvine32 library.

###  Technologies Used
- MASM (Microsoft Macro Assembler)
- Irvine32 Library
- x86 Assembly Language
- Windows Console
